# 🧠 Fraud Detection AI
### Yapay Zeka ile Dolandırıcılık Tespiti

Bu proje, makine öğrenmesi ve veri analizi teknikleri kullanarak finansal işlemlerdeki dolandırıcılık faaliyetlerini tespit etmeyi amaçlayan bir yapay zeka sistemidir.

---

## 🎯 Proje Amacı

Bu projenin amacı:

- Büyük veri setlerini analiz etmek
- Şüpheli finansal işlemleri tespit etmek
- Makine öğrenmesi algoritmaları ile dolandırıcılık tahmini yapmak
- Finansal güvenlik için akıllı bir analiz sistemi geliştirmek

---

## 🛠 Kullanılan Teknolojiler

Projede kullanılan teknolojiler:

- 🐍 Python
- 📊 Pandas
- 🔢 NumPy
- 🤖 Scikit-Learn
- 📉 Matplotlib
